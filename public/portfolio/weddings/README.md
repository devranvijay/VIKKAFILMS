# Weddings Portfolio

Drop your wedding shoot images here.

**Naming convention:** `project-name-01.jpg`, `project-name-02.jpg`
**Supported formats:** .jpg, .jpeg, .png, .webp
**Recommended size:** min 1200px wide, under 2MB each

After adding images, register them in:
  app/data/portfolio.ts

Example entry:
```ts
{
  id: 10,
  category: "Weddings",
  title: "Sharma Wedding",
  description: "Luxury destination wedding · Udaipur",
  src: "/portfolio/weddings/sharma-wedding-01.jpg",
  featured: true,
}
```
