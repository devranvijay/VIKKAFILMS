# Commercial Portfolio

Drop your commercial shoot images here.

**Naming convention:** `brand-name-01.jpg`
**Supported formats:** .jpg, .jpeg, .png, .webp
**Recommended size:** min 1200px wide, under 2MB each

After adding images, register them in:
  app/data/portfolio.ts

Example entry:
```ts
{
  id: 20,
  category: "Commercial",
  title: "Chronos Series",
  description: "Luxury timepiece campaign · Mumbai",
  src: "/portfolio/commercial/chronos-01.jpg",
  featured: true,
}
```
